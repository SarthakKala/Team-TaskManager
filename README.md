# Team Task Manager

A full-stack web app for managing team projects and tasks with role-based access control.

## Tech Stack

- **Frontend**: React (Vite) + Tailwind CSS — Black & White Retro Design
- **Backend**: Node.js + Express.js
- **Database**: PostgreSQL via [Neon](https://neon.tech)
- **ORM**: Prisma
- **Auth**: JWT + bcrypt
- **Deployment**: Railway

## Features

- JWT Authentication (Signup / Login)
- Role-based access: Admin & Member
- Project creation and management
- Task creation, assignment, and status tracking
- Dashboard with stats (Total, Done, In Progress, Overdue)
- Responsive retro black & white UI
